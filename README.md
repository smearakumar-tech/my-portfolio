# Smeara Kumar — Personal Portfolio Website

A modern, responsive, production-ready personal portfolio website built with **HTML5**, **CSS3**, **Bootstrap 5**, and **Vanilla JavaScript**. Deployed live via GitHub Pages.

## ✨ Features

- ✅ Fully responsive — mobile, tablet, laptop, desktop
- ✅ Dark premium UI with glassmorphism cards
- ✅ Gradient accents (cyan, indigo, purple)
- ✅ Hero section with **typing animation** (5 rotating roles)
- ✅ Sticky glassmorphic navbar with active highlighting
- ✅ Smooth scrolling with **IntersectionObserver** scroll-reveal
- ✅ Animated blob avatar with morph effect
- ✅ 9 skill icon cards with animated progress bars
- ✅ 6 responsive project cards with GitHub & Live Demo buttons
- ✅ Professional contact form with validation & success state
- ✅ Download Resume button
- ✅ Scroll-to-top floating button
- ✅ Social media links (GitHub, LinkedIn, Email)
- ✅ Animated background orbs
- ✅ Poppins Google Font

## 🚀 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/smeara-portfolio/smeara-portfolio.git
cd smeara-portfolio
```

### 2. Open locally
Simply open `index.html` in your browser:
```bash
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

No build tools or npm required — this is a pure static website!

## 🔧 Customization Guide

To make this portfolio your own:

1. **Personal Info** — Update `index.html` with your name, bio, email, location, and social links.
2. **Resume** — Replace `assets/resume/Smeara_Kumar_Resume.pdf` with your own PDF.
3. **Avatar** — Replace `assets/images/avatar.png` with your photo.
4. **Projects** — Edit the project cards in `index.html` with your actual project details and GitHub/live links.
5. **Skills** — Adjust the progress bar `data-width` values in the Skills section.
6. **Colors** — Modify `--g1`, `--g2`, `--g3` CSS variables in `style.css` to change the gradient palette.
